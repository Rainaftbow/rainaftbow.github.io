安装过程

1、官网下载对应版本的CLI https://github.com/AlistGo/alist/releases

2、在exe目录下输入 alist.exe admin set 你的密码 初始化管理员账号

3、alist.exe server 启动本地服务

4、打开浏览器地址栏输入 localhost:5442 打开管理页面 （图中所示是我配置好的，第一次打开啥都没有）

5、添加云盘 官方文档很详细（yin wei wo lan） https://alist.nn.ci/zh/guide/drivers/common.html

Tip：无服务器部署替代方法

由于之前买的腾讯云到期，无法部署到云，每次使用都要在本地启动服务，很麻烦。

通过添加开机自启可解决这个问题

1、在有alist.exe的目录下创建一个bat文件

2、然后写入内容 powershell Start-Process -WindowStyle hidden -FilePath .\alist.exe server

3、再对着bat文件右键创建 “桌面快捷方式”

4、win+r 打开运行 输入 shell:startup 回车后会打开一个目录, 把创建的快捷方式拖进去
