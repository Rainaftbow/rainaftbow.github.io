安装过程

1、官网下载对应版本的CLI https://github.com/AlistGo/alist/releases

2、在exe目录下输入 alist.exe admin set 你的密码 初始化管理员账号

3、alist.exe server 启动本地服务

4、打开浏览器地址栏输入 localhost:5244 打开管理页面 （图中所示是已经配置好的云盘，请忽略）

5、添加云盘 官方文档很详细（yin wei wo lan） https://alist.nn.ci/zh/guide/drivers/common.html

云盘挂载

为能够更方便使用，可将云盘挂载到本地。

云盘本地挂载是一种将云存储服务直接映射到本地文件系统的方法

步骤：
1、安装RaiDrive
RaiDrive下载地址 https://www.raidrive.com/download

2、安装完后，打开软件可以看到这个界面，点击添加

3、按图中标注配置

4、最后点击连接，连接成功后挂载的本地文件夹会弹窗
（8 EB = 8,000,000,000 GB）
（是的，你没看错😂）

无服务器部署替代方法

由于之前买的腾讯云到期，无法部署到云，每次使用都要在本地启动服务，很麻烦。

通过添加开机自启可解决这个问题

1、在有alist.exe的目录下创建一个vbs文件

2、然后写入内容

1
2
3

Dim ws  
Set ws = Wscript.CreateObject("Wscript.Shell")
ws.run "cmd /c .\alist.exe server",vbhide

注意把alist.exe路径改为你的路径!
